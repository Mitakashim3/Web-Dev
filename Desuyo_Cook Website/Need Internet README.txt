# Cook Web

This is a one-page parallax scrolling website template designed to showcase a cooking recipe for rice. The website features a parallax scrolling effect, responsive design, and nutritional information table.

## Getting Started

To run the website locally, follow these steps:

1. **Download the zip file**: Download the zip file containing the website files.

2. **Extract the files**: Extract the contents of the zip file to a folder on your computer.

3. **Open the folder**: Navigate to the folder where you extracted the files.

4. **Open the HTML file**: Double-click the `index.html` file to open it in a web browser.

5. **View the website**: The website should now load in your web browser. You can navigate through the sections by scrolling or using the navigation links in the header.

## Requirements

- Web browser (Chrome, Firefox, Safari, etc.)
- Internet connection (for loading fonts and photos)
